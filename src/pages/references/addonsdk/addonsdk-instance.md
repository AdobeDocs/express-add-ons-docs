# addOnUISdk.instance

Represents the currently running add-on instance. This object is used to provide access to the `clientStorage` and `manifest` objects. See the [Storing and Retrieving Client Side Data](../../guides/develop/how_to/local_data_management.md) use case implementation and [Manifest](../manifest) reference for more details.
