# addOnUISdk.constants

A set of constants used throughout the add-on SDK. These constants are equal to their variable name as a string value, ie: for the `ButtonType` constant, `primary` has a value of "primary".

## Constants

| ---- | ---- | ----------- |

| `BleedUnit` | `string` | Units for the page bleed.&lt;ul&gt;&lt;li&gt;**"in" (`Inch`)** &lt;br&gt; Inch units.&lt;/li&gt;&lt;li&gt;**"mm" (`Millimeter`)** &lt;br&gt; Millimeter units.&lt;/li&gt;&lt;/ul&gt; |
| `ButtonType` | `string` | The type of the button pressed in a dialog.&lt;ul&gt;&lt;li&gt;**primary** &lt;br&gt; Primary button pressed.&lt;/li&gt;&lt;li&gt;**secondary** &lt;br&gt; Secondary button pressed.&lt;/li&gt;&lt;li&gt;**cancel** &lt;br&gt; Cancel button pressed.&lt;/li&gt;&lt;li&gt;**close** &lt;br&gt; Dialog closed via ESC or close(X) button.&lt;/li&gt;&lt;/ul&gt; |
