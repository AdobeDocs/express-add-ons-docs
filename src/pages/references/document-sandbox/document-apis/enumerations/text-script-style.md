# Enumeration: TextScriptStyle

Represents a text script style.

## Enumeration Members

| Enumeration Member | Value | Description |
| ------ | ------ | ------ |
| `none` | `0` | text appears at the standard baseline |
| `superscript` | `1` | text appears above the baseline |
| `subscript` | `2` | text appears below the baseline |
